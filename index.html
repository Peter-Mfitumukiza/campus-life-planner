About

Purpose

Campus Life Planner helps students organize their academic and personal tasks efficiently. Track assignments, study sessions, meetings, and events with duration tracking, tags, and powerful regex search.

Features

  • Add, edit, and delete tasks with due dates and durations
  • Organize tasks with customizable tags
  • Advanced regex search with pattern highlighting
  • Visual statistics dashboard with weekly targets
  • Data persistence with localStorage
  • Import/Export JSON for backup and migration
  • Fully accessible with keyboard navigation
  • Responsive design for mobile, tablet, and desktop
